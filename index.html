Tools inside the hub

The landing page introduces four lightweight modules: a stopwatch, a quiz, a to-do manager and the landing itself as the entry experience. Here I only built the structure – the other modules live in separate tasks.

01 · Entry

Landing page

Sets the tone, explains the idea and gives a single place to understand what the toolkit offers.

02 · Time

Stopwatch module

A focused time tracker used to time short study or coding sessions. (Implemented in Task 02.)

03 · Learning

Quiz module

A small quiz that plays with question flow and instant feedback. (Implemented in Task 03.)

04 · Tasks

To-do module

A daily task list with filters and simple priorities. (Implemented in Task 04.)

About this task

For this task I wanted the page to feel like a real product landing, not a “lab exercise”. So I focused on a simple hierarchy: a clear hero, one block explaining the tools and one block explaining the intention behind the project.
